# Flutter Provider State Management

Welcome to the repository for the **State Management Providers** course, part of the *Mastering Flutter in Arabic 2021* series by **Wael Abo Hamza**. 

This repository contains the source code, examples, and projects built throughout the course to understand and master the Provider package for state management in Flutter.

## 📺 Course Link
You can watch the full course playlist on YouTube here:
[State management Providers [Mastering Flutter In Arabic] 2021](https://youtube.com/playlist?list=PL93xoMrxRJIviJiC76oO5aV8bDp2s3OrA)

## 🎯 Objectives
- Understand the core concepts of State Management in Flutter.
- Learn how to implement the `provider` package effectively.
- Build reactive, scalable, and clean Flutter applications.
- Separate business logic from the UI.

## 🚀 Getting Started

### Prerequisites
To run this project, you will need to have the following installed on your machine:
- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- An IDE such as [VS Code](https://code.visualstudio.com/) or [Android Studio](https://developer.android.com/studio)
- A physical device or an emulator (Android/iOS) to test the application.
